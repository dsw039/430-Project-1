Book API Documentation

Add Book(Post)

Supported Formats

application/json

application/x-www-form-urlencoded

Body Params

"title" Type: "string" - Name of the Book

"author" Type: "string" - Name of the Author

Returns

A status code indicating a successful or unsuccessful addition

Examples

Request: api/addBook

Body: {title,author}

Result: {"message":"Created Successfully"}

Request: api/addBook

Body: {title}

Result: {"message":"author and title are both required.","id":"missingParams"}

N/A

Get Book(GET)

Query Params

"title" Type: "string" - Name of the Book

Returns

A JSON array of Book Objects with the same title(usually an array of just 1 book)

Examples

Request: api/getBook?filteredBooks=Things-Fall-Apart

Results: [{"author":"Chinua Achebe","country":"Nigeria","language":"English","link":"https://en.wikipedia.org/wiki/Things_Fall_Apart\n","pages":209,"title":"Things Fall Apart","year":1958,"genres":["Historical Fiction","Postcolonial Literature"]}]

Request: api/getBook?filteredBooks=FakeBook

Results: {"message":"Book Not Found"}

Get Genre(GET)

Query Params

"genre" Type: "string" - Genre of the Book

Returns

A JSON array of Book Objects with, 1, a listed genre and 2, a genre matching the requested genre

Examples

Request: api/getGenre?filteredBooks=Modernism

Results: {"books":[{"author":"a1","country":"c1","language":"l1","link":"ll1","pages":1,"title":"t1","year":1,"genres":["Modernism"]},{"author":"a2","country":"c2","language":"l2","link":"ll2","pages":1,"title":"t2","year":1,"genres":["Modernism"]},{"author":"a3","country":"c3","language":"l3","link":"ll3","pages":1,"title":"t3","year":1,"genres":["Modernism"]},...}

Request: api/getGenre?filteredBooks=modernism

Results: {"message":"Genre Not Found"}

Get Language(GET)

Query Params

"language" Type: "string" - Language of the Book

Returns

A JSON array of Book Objects with the same listed language

Examples

Request: api/getGenre?filteredBooks=Japanese

Results: [{"author":"Yasunari Kawabata","country":"Japan","language":"Japanese","link":"https://en.wikipedia.org/wiki/The_Sound_of_the_Mountain\n","pages":288,"title":"The Sound of the Mountain","year":1954},{"author":"Murasaki Shikibu","country":"Japan","language":"Japanese","link":"https://en.wikipedia.org/wiki/The_Tale_of_Genji\n","pages":1360,"title":"The Tale of Genji","year":1006}]

Request: api/getGenre?filteredBooks=Martian

Results: {"message":"Language Not Found"}
